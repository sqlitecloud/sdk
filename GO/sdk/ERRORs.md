## Error Reason

| Code | Alternative | Explanation |
|------|-------------|-------------|
| 0    | -            | Value is NULL/NIL/ZERO |
| 1    | -            | Value is to small |
| 2    | -            | Value is to large |
| 3    | 1 + 2       | Value is out of bounds |
